== Description ==
Wiloke Most Popular Posts Widget is a free WordPress plugin, it comes with lots of features such as Order By: The most commented / visited, Set Number Of Posts, allows to you to display perfect list of most popular posts on sidebar with 5 kind of style.

<a href="https://wiloke.com/product/wiloke-most-popular-posts-widget/"><img src="https://blog.wiloke.com/wp-content/uploads/2015/10/widget.jpg" alt="Wiloke Most Popular Posts Widget" /></a>

<h3><strong>Key Features & Options</strong></h3>
<ul>
	<li>Order By: The most commented / visited</li>
	<li>Set Number Of Posts</li>
	<li>Show the posts that are the most Visited/Commented in: This Week, This Month, This Day, Last 7 days ago, Last 30 days ago and Last 1 day ago.</li>
	<li>Style: We offer you five styles:
		<ul>
			<li>Style1: First Larger Then Listing. Show all of the posts with post thumbnails.</li>
			<li>Style2: First Larger Then Listing. The posts listing don’t show the post thumbnails.</li>
			<li>Style3: The posts listing. Show all of the posts with post thumbnails.</li>
			<li>Style4: Unordered List. Don’t show the post thumbnails.</li>
			<li>Style4: Ordered List. Don’t show the post thumbnails.</li>
			<li>We are using Wiloke Most Popular Posts Widget for this site: <a href="http://titan.wiloke.com/" target="_blank">http://titan.wiloke.com/</a></li>
		</ul>
	</li>
</ul>
